# example gps sensors

Some android gps sensors example.

Example includes:

- maps api
- gps fetching
- gps normalization with Kalman
- steps tracking
- calories tracking
- session example and packing paradygm
- local persistence

## Requirements
* Java 11
* Android SDK
* Gradle

## Building

Build tool is gradle

### Assemble
Run `./gradlew assemble`

### Run unit tests
Run `./gradlew test`

### Run ui tests
Run `./gradlew connectedCheck`

### Run lint
Run `./gradlew lint`
